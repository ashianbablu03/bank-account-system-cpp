# Bank Account System (C++)

A C++ banking simulation that demonstrates inheritance and polymorphism through two account types: Checking and Savings.

## Features
- Checking account with overdraft protection
- Savings account with interest accrual and monthly withdrawal limits
- Deposit, withdraw, and account-to-account transfers
- Full transaction history log per account

## Concepts Demonstrated
- Object-Oriented Programming (Inheritance, Polymorphism)
- Abstract base classes and virtual functions
- STL Vectors for transaction history
- Encapsulation and data protection

## How to Run
```bash
g++ bank_account.cpp -o bank_account
./bank_account
```

## Technologies
- Language: C++
- Compiler: g++ (C++17)
